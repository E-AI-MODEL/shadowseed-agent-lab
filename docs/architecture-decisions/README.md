# Architecture Decision Records

This directory records architecture decisions for `shadowseed-agent-lab`.

Architecture decisions are short, durable records. They explain the chosen boundary, the reason for it, and the relation to upstream `E-AI-MODEL/shadowseed`.

## Index

- [0001: Agent lab boundary around upstream SSL](0001-agent-lab-boundary.md)
- [0002: RAG is evidence, not influence](0002-rag-as-evidence-not-influence.md)

## Rules

- Keep decisions compact.
- Prefer explicit boundaries over broad claims.
- Do not restate the full SSL theory when upstream owns it.
- Add a new record when a decision changes the agent/RAG integration boundary.
